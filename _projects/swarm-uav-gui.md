**Duration:** Aug 2024 – Dec 2024  
**Organization:** VNU University of Engineering and Technology (VNU-UET), Hanoi, Vietnam  
**Supervisor:** Nguyen Quang Nha, UET-VNU

## Overview

Designed a graphical user interface and simulation framework for monitoring and coordinating swarm UAVs in search-and-rescue scenarios. The system allows researchers to visualize and test swarm coordination algorithms before real-world deployment.

## Key Contributions

- Designed an intuitive GUI for real-time swarm UAV state visualization
- Built a simulation framework for testing coordination algorithms in virtual environments
- Collaborated with VNU-UET researchers on swarm behavior modeling, swarm control algorithms, and performance evaluation
