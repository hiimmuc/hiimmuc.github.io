**Duration:** Apr 2021 – May 2021  
**Organization:** Hanoi University of Science and Technology (HUST), Hanoi, Vietnam  
**Supervisor:** Nguyen Quang Nha, UET-VNU

## Overview

Developed a deep learning–based Automatic License Plate Recognition (ALPR) system and integrated it with embedded parking gate hardware via the RS-485 communication protocol for a complete smart parking solution.

## Key Contributions

- Built and trained the ALPR model for Vietnamese license plates
- Integrated the system with embedded devices using RS-485 serial communication
- Designed end-to-end pipeline from camera capture to gate control signal

## Technologies

`Deep Learning` `ALPR` `Python` `RS-485 Protocol` `Embedded Systems`
